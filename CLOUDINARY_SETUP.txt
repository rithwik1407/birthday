CLOUDINARY VIDEO UPLOAD SETUP - COMPLETE GUIDE
===============================================

STEP 1: CREATE CLOUDINARY ACCOUNT
==================================
1. Go to https://cloudinary.com
2. Click "Sign Up Free"
3. Fill in your details:
   - Email
   - Password
   - Name
4. Click "Create Account"
5. Verify your email
6. Login to your Cloudinary dashboard

STEP 2: GET YOUR CLOUDINARY CREDENTIALS
========================================
1. Go to https://dashboard.cloudinary.com
2. Look at the top of the page - you'll see your "Cloud Name"
3. Copy your Cloud Name (looks like: abc123xyz)
4. Go to Settings → API Keys
5. You'll see:
   - Cloud Name (already have this)
   - API Key
   - API Secret
6. Copy these values

STEP 3: UPDATE YOUR .env FILE
==============================
Open your .env file and add/update these lines:

NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=your_cloud_name_here
NEXT_PUBLIC_CLOUDINARY_UPLOAD_PRESET=your_upload_preset_here
CLOUDINARY_API_KEY=your_api_key_here
CLOUDINARY_API_SECRET=your_api_secret_here

Example:
NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=dxyz1234
NEXT_PUBLIC_CLOUDINARY_UPLOAD_PRESET=sweety_birthday
CLOUDINARY_API_KEY=123456789abcdef
CLOUDINARY_API_SECRET=xyz789abc123def

STEP 4: CREATE UPLOAD PRESET
=============================
1. Go to https://dashboard.cloudinary.com
2. Click "Settings" (gear icon)
3. Go to "Upload" tab
4. Scroll down to "Upload presets"
5. Click "Add upload preset"
6. Fill in:
   - Name: sweety_birthday (or any name you want)
   - Unsigned: Toggle ON (important!)
   - Resource type: Video
   - Folder: birthday_videos (optional)
7. Click "Save"
8. Copy the preset name

STEP 5: UPDATE .env WITH UPLOAD PRESET
=======================================
Update your .env file with the preset name you created:

NEXT_PUBLIC_CLOUDINARY_UPLOAD_PRESET=sweety_birthday

STEP 6: INSTALL CLOUDINARY PACKAGE
===================================
Run this command in your terminal:

npm install next-cloudinary

TROUBLESHOOTING:
================
If upload fails:
- Check Cloud Name is correct
- Check Upload Preset exists and is "Unsigned"
- Check .env file is saved
- Restart the dev server: npm run dev

If video doesn't play:
- Check video format is supported (MP4, WebM, OGG)
- Check video file size (Cloudinary has limits)
- Check video URL in Cloudinary dashboard
